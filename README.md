# 苏州大学 2020级 电气工程及其自动化
# 本科课程作业合集

## 仓库用途
本仓库收录 2020 级电气工程及其自动化专业的本科大作业与课程项目，供回顾、学习与复现使用。每个子目录尽量保留源代码、实验报告、必要的数据与说明文档。

## 收录内容
- ELEA3063 新一代人工智能原理  
- ELEA3037 工程电磁场  
- ELEA3040 电机与拖动  
- ELEA2015 电力电子技术  
- 00272005 程序设计及应用 (C语言)

## 目录结构
每门课程目录通常包含：
- /code — 源代码与仿真文件  
- /report — 实验报告或课程设计文档  
- /data — 样例数据或测试用例  
- README.md — 该课程的简要说明

## 使用方法
1. 克隆或下载仓库到本地：git clone <仓库地址>  
2. 进入感兴趣的项目目录，查看 README 或 report 获取运行/复现实验所需步骤。  
3. 如发现可补充的资料或修正，可提交 issue 或 pull request。
